# Security Policy

## Supported versions

Only the latest public release is supported.

## Reporting

Open a private security advisory on GitHub if available, or open an issue without publishing sensitive data.

Do not include:

- Public IP addresses.
- Home Assistant tokens.
- Serial numbers.
- Full network topology.
- Private logs with personal data.

## Scope

Security reports should relate to this integration code, diagnostics, services, or unsafe data exposure.
